voting-by-age-orig.csv contains information about Reported Voting and
Registration of the Total Voting-Age Population by Age, for States:
November 2008.

voting-race-sex-orig.csv contains the same information by sex, race, and
Hispanic origin.

voting-race-sex.csv and voting-by-age.csv contain this information in
processed format.

state-pop-fromcensus.gov.csv contains data about the population of
different states.

statepop.csv contains the 2012 census populations from census.gov.

state-demographics-race contains the racial breakdown of each state (from
Wikipedia I think).

state-demographics.age.xls contains the age breakdown of each state (from
census.gov) in 2010.

state-demographics-age.csv and state-demographics-race.csv contain the 2010
census data for race and age.
